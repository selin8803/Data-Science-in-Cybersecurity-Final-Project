# Data-Science-in-Cybersecurity-Final-Project
# Phishing Website Detection using Machine Learning Techniques

## Project Description

The objective of the project is to reproduce and critically evaluate a machine learning solution for phishing website detection. Several machine learning models were trained and compared using a phishing website dataset.

The evaluated models include:

- Decision Tree
- Random Forest
- Multilayer Perceptron (MLP)
- XGBoost
- Support Vector Machine (SVM)

The models were evaluated using multiple performance metrics, including Accuracy, Precision, Recall, F1-Score, ROC-AUC, Matthews Correlation Coefficient (MCC), and Confusion Matrix.
